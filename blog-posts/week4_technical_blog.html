Enumerable#map

Enumerable is a very interseting module within Ruby. Ruby's official documentation describes enumerable as a "mixin (that) provides collection classes with traversal and searching methods and with the ability to sort." The class must provide a method each, which gives you successive members of the collection. In short, enumberables enable one to repeat actions with a class.

The enumerable we're focused on right now is Enumerable#map. Enumerable#map creates a new array by performing some operation on every item in the given collection. Map and collect are the same - you can use either. Map, however, is shorter and draws a comparison to functional mapping.

At its most basic level, enumerable#map modifies each character in a consistent way:

[1, 2, 3, 4].map {|number| number*2}

=>[2, 4, 6, 8]

["Bat", "Aqua", "Super"].map {|adjective| adjective+"man"}

=>["Batman", "Aquaman", "Superman"]
