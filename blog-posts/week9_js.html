Cheat Sheet - Javascript

Javascript Basics

Types of Characters

Strings - var bananas = "I have some bananas"; var oranges = 'Don\'t have oranges'; Continaed in double or single quotes
Numbers - var numberOfBanas = 5; var numberOfOranges = 0;
Boolean - var haveBananas = true; var haveOranges = false; Value of true or false, no quotes
Undefined - var strawberries; No value assigned to it yet
Null - var pumpkins = null; No quotes; a purposely empty value (not the same as 0 or undefined)

Functions

console.log('Hi!!');
}
sayHi(); //Call the function
